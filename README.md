# Safe Bite - "Unwrap What's In Your Food"

Safe Bite is a web-based accessibility-first health tool that allows users to understand what's really inside their food. By scanning any products barcode, users recieve a clear breakdown of ingredients, including animal derived components, allergens, and dietary confilicts that are often confine behind chemical names. 

## Mission
Our mission is to ensure that food transparency becomes a universal right. Empowering every person, regardless of ability, language, or dietary needs, to understand what they consume and make choices that honor their health, identity, and values.

## Key Features
- Barcode Scanner: Real‑time product scanning using camera-based detection powered by html5-qrcode engine with custom scan windows and frame processing.
- Ingredient Intelligence (Gemini AI): AI‑driven parsing that translates complex chemical ingredients into clear, human‑readable explanations.
- Accessibility (ElevenLabs Audio Agent): Seamless text‑to‑speech guidance that reads instructions and ingredient results aloud for enhanced accessibility.
- Personalized User Profiles: Supabase‑backed profiles that store allergies, health conditions, and dietary preferences for automatic conflict detection.
- Mobile Friendly Web App: Responsive Vite‑powered interface with an Express backend, optimized for fast, intuitive scanning and smooth onboarding across all devices.

## Geting Started
#### 1. Prerequisites
- React (v18+)
- Express (v4+)
- Next (v14+)
- Supabase (v2)
- npm Packages
- Internet access (for Supabase, ElevenLabs, and external API calls)

#### 2. Installation
```
git clone [https://github.com/shenleii10/ellehacks26](https://github.com/shenleii10/ellehacks26)
cd ellehacks26
npm install
```
#### 3. Run the Application
Backend
```
cd safe-bite
npm run dev
```
Frontend
```
cd frontend\src
npm run dev
```
Open http://localhost:3000 to use the app.
